BLACK COFFEE - MODULO POWERSHELL (Windows)
==========================================

Carpeta del proyecto:
  C:\Users\LENOVO\Documents\BlackCoffeeAdmin

CARGAR MODULO (copiar y pegar en PowerShell):
  cd C:\Users\LENOVO\Documents\BlackCoffeeAdmin
  Import-Module .\scripts\BCA\BCA.psm1 -Force

COMANDOS DISPONIBLES:
  Open-BCAEnlaces          -> Abre todas las paginas web
  Install-BCAProject       -> Instala todo (Node, Firebase, secretos)
  Start-BCASetup -OpenLinks -> Descarga/actualiza + iconos + abrir
  Get-BCAConfig            -> Ver rutas y URLs
  Sync-BCARepository       -> git pull

CLI RAPIDA:
  .\scripts\bca.ps1 abrir
  .\scripts\bca.ps1 instalar
  .\scripts\bca.ps1 setup

DOBLE CLIC (sin PowerShell):
  ABRIR-ENLACES.bat
  INSTALAR-Y-ABRIR.bat

Iconos en Escritorio:
  BCA - Abrir enlaces
  BCA - Instalar y abrir
  BCA - Consola PowerShell

Documentacion: scripts\BCA\README.md
